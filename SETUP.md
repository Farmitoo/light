
# Installation
To install all the dependencies
```
composer install
```

If you use yarn 
```
yarn install
```
or npm
```
npm install
```

# Build Assets

## with yarn 
If you need to build assets (prod)
```
make assets-build
```
or to watch (dev)
```
make assets-watch
```

## with npm
 build assets (prod)
```
npm run build
```
or watch with (dev)
```
npm run watch
```

# Run server and navigate
Run server
```
make server
```

Then navigate to http://localhost:8080/
